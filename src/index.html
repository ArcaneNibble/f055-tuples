What is this?

There might come a time when a microcontroller or other embedded project needs a globally-unique tuple of two 16-bit integers. At one point, this website contained a registry of 16-bit tuples where the first element was the value 0xf055. Unfortunately, that previous website has since ceased to exist. This new website is an attempt to both revive the old registry of tuples as well as to link to other places where open hardware projects can obtain 16-bit tuples beginning with numbers other than 0xf055.
